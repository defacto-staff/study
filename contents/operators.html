Operators are used to perform operations on variables and values.

PHP divides the operators in the following groups:

  • 代数演算子
  • 代入演算子
  • 比較演算子
  • 加算子/減算子
  • 論理演算子
  • 文字列演算子
  • 配列演算子
  • 三項演算子

三項演算子

条件演算子として "?:"（あるいは三項）演算子があります。

例: デフォルト値を設定する

<?php
// 三項演算子の使用例
$action = (empty($_POST['action'])) ? 'default' : $_POST['action'];

// 上記は以下の if/else 式と同じです。
if (empty($_POST['action'])) {
    $action = 'default';
} else {
    $action = $_POST['action'];
}
?>

(expr1) ? (expr2) : (expr3) という式は、式1 が TRUE の場合に 式2 を、 式1 が FALSE の場合に 式3 を値とします。

以降では、三項演算子のまんなかの部分をなくすこともできるようになりました。 式 expr1 ?: expr3 の結果は、expr1 が TRUE と同等の場合は expr1、 それ以外の場合は expr3 となります。
