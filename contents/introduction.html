PHPはサーバー上で実行されます。

PHPって？

  • PHPは、「PHP: Hypertext Preprocessor」の略語です。
  • PHPは、広く使われている、オープンソースのスクリプト言語です。
  • PHPのスクリプトは、サーバー上で実行されます。
  • PHPは、ダウンロードするのも使うのもタダです。

PHP is an amazing and popular language!

It is powerful enough to be at the core of the biggest blogging system on the web (WordPress)!
It is deep enough to run the largest social network (Facebook)!
It is also easy enough to be a beginner's first server side language!

PHPファイルって？

  • PHPのファイルには、テキスト、HTML、CSS、JavaScript、そしてPHPのコードを記述することができます。
  • PHPのコードは、サーバー上で実行され、その結果がプレーンなHTMLとしてブラウザに返ってきます。
  • PHPファイルの拡張子は「.php」です。

PHPはどんなことができるの？

  • PHPは、動的なページのコンテンツを生成することができます。
  • PHPは、サーバー上で、ファイルを作成したり、開いたり、読み込んだり、書き込んだり、削除したり、閉じたりすることができます。
  • PHPは、フォームのデータを集めることができます。
  • PHPは、cookie（クッキー）を送ったり受け取ったりすることができます。
  • PHPは、データベースにデータの追加・削除・修正をすることができます。
  • PHP can be used to control user-access
  • PHPは、データを暗号化することができます。

PHPにできることは、HTMLを出力することに限られているわけではありません。画像やPDFファイル、そしてまたFLASHの動画なども出力することができます。XHTMLやXMLのような、どんなテキストもまた出力することができます。

どうしてPHPなの？

  • PHP runs on various platforms (Windows, Linux, Unix, Mac OS X, etc.)
  • PHP is compatible with almost all servers used today (Apache, IIS, etc.)
  • PHP supports a wide range of databases
  • PHP is free. Download it from the official PHP resource: www.php.net
  • PHP is easy to learn and runs efficiently on the server side
